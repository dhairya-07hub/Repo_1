Fundamental/Financial Computer Applications (FCA) – Course Syllabus

Semester 1

  • Basic Concepts and Fundamentals of Computers
  • Computer Hardware & Software Overview
  • Operating System Basics (Windows/Linux)
  • English Typing (Basics)
  • MS Word (Formatting, Editing, Saving Documents)
  • Printing and Scanning Basics
  • MS Excel (Spreadsheets – Basic Functions)
  • MS PowerPoint (Basic Presentations)
  • Introduction to Internet & Email Usage
  • Project/Practical Work (Word, Excel, Internet)

Semester 2

  • Database Introduction (MS Access or equivalent)
  • Tally/Accounting Software Basics (Data Entry, GST Introduction)
  • Fundamental Banking & Tax Form Filling Tools
  • Financial Data Analyzing Tools
  • Cyber Security Essentials
  • Communication & Collaboration Tools
  • Web and Multimedia Basics
  • Application/Project Work (Accounting/Database)
